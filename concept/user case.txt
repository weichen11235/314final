User Story:
User wants to read some jokes and have fun!

Assumption:
User will enter the events page from a url link 

Steps:
1. User loads events page
2. User click a tab either single joke or multiple joke
3. The contents that are specific to that tab then load into that
4. in the single joke tab, user can click button to get another joke.
in the multiple joke tab, user can click button to choose joke categories and get multiple joke.
